# Проект React-mesto-auth

Продолжение проекта Mesto, введены новые фичи- авторизация и регистрация, теперь приложение взаимодействует сразу с двумя API: авторизации и API редактирования пользователя и получения карточек.

Проект дополнен роутами авторизации и регистрации пользователя, которые защищают main рут- для доступа к нему необходим статус авторизированного пользователя. Реализована работа с локальным хранилищем токена- при работе с сайтом токен сохраняется, при повторном визите пользователи вновь должны пройти авторизацию.

## В данном проекте используется стек Frontend-технологий: JavaScript, Webpack, HTML, CSS, и React
